# R19 — Genitive Absolute Scholarship

Substrate for canon §3.10a (R19 Genitive Absolute Always Own Line). The operational rule lives in the canon entry; this document carries the grammatical-grounding citations and the cataphoric-vs-anaphoric framework §1.1 interaction.

## Burton + Wallace + BDF grounding

- **Burton §428–429** — genitive absolute as detached frame; agent in genitive, predicate via genitive participle.
- **Wallace pp.654–665** — modern comprehensive treatment of the construction; covers cataphoric (forward-looking) and anaphoric (backward-looking) uses with NT case studies.
- **BDF §423** — philological grounding; primary scholarly source for the construction as a classical-Greek category in Koine.

The literal sense of "absolute" is "set apart" / "loosened from" (Latin *absolutus* < *ab + solvere*). The genitive case marks the construction's detachment from the main clause's nominative-marked subject — the gen abs has its own subject and its own (participial) predicate, neither of which integrates into the main clause's grammar.

## Why R19 fires regardless of cataphoric / anaphoric direction

R19 is a colometric rule: it governs LINE PLACEMENT, not ATU status. A gen abs always earns its own line because the construction is structurally a separable frame. Whether that frame functions cataphorically (introducing the next clause) or anaphorically (back-referencing the prior clause) is a downstream discourse-function question handled at framework §1.1's bidirectional ATU test.

### Cataphoric example (forward-looking frame)

**Matt 1:20** — `Ταῦτα δὲ αὐτοῦ ἐνθυμηθέντος,` / `ἰδοὺ ἄγγελος κυρίου …` — "And while he was contemplating these things, behold, an angel of the Lord…" The gen abs frames the angel's appearance; the main clause follows on the next line.

The cataphoric gen abs passes the standalone-ATU test in the cataphoric direction (it introduces something).

### Anaphoric example (backward-referencing frame)

**Acts 4:1** — `Λαλούντων δὲ αὐτῶν πρὸς τὸν λαόν,` standing as own line. The `αὐτῶν` refers back to Peter and John who were just speaking; the gen abs back-references the prior context.

The anaphoric gen abs *fails* the standalone-ATU test in the cataphoric direction but *succeeds* in the anaphoric direction. Per framework §1.1's bidirectional test, this is still a legitimate ATU (the anaphoric direction satisfies the test); R19 still mandates own-line.

## Acts 1:9 showcase (canon §1 / framework §1.1)

```
Καὶ ταῦτα εἰπών,           ← circumstantial participle + ταῦτα anaphoric reference
βλεπόντων αὐτῶν             ← genitive absolute (camera-shift to witnesses)
ἐπήρθη                      ← main clause
καὶ νεφέλη ὑπέλαβεν αὐτὸν   ← coordinate main clause
ἀπὸ τῶν ὀφθαλμῶν αὐτῶν.     ← prepositional adjunct
```

The gen abs `βλεπόντων αὐτῶν` is its own ATU: a separate camera angle on the witnesses, distinct from the main verb's action of ascension. Severing it across lines would either submerge it into the speech-act frame (line 1) or into the ascension predication (line 3) — both losses of structural information.

## Boundary with R20 (circumstantial / supplementary / attributive participles)

R19 covers only the absolute construction (anarthrous genitive participle + agreeing genitive subject). Non-gen-abs participles fall under R20:

- **Circumstantial participle** (default merge with main verb): adverbial frame; dependent.
- **Supplementary participle** (split when independent predication via ellipsis): second-predication test.
- **Attributive participle** (merge as noun modifier): articular, modifies a noun.

R19 fires *first*; R20 handles the residue.

## Interaction with R28-ext (Speech-Act After Frame)

R28-ext fires when a substantive frame precedes a finite speech verb on the same line. If the frame is a genitive absolute, R19 has already split it to its own line — R28-ext is redundant in that case (per R28-ext Exclusion 4). The R19 → R28-ext yield is one-way: R19 governs first; R28-ext does not layer on top of an R19 application.

## Interaction with M4-GNT-1 (Subject-Orphan Predicate Completion)

M4-GNT-1 universal exclusion #11 names R19 as a precedence yield: when line B begins with a gen abs, R19 fires first; M4-GNT-1 does not merge. This is a one-way yield matching the framework Tier 2 vs Tier 4 ordering.
