# R8 — Discourse-Linguistic Grounding for Framing Devices

Scholarly substrate for canon §3.3 (R8 Framing Devices Attach). The operational rule (lead-content, never-line-final) and its closed list (`FRAMING_DEVICES`) live in the canon entry; this document carries the per-marker discourse-function nuance and the Runge / Levinsohn citations that ground the closed list.

## Per-marker discourse function

The colometric rule — *framing devices lead their content; never line-final* — is uniform across all members of `FRAMING_DEVICES`. But the **discourse direction** varies marker by marker. The loose gloss "introduce what follows" is over-extended for one member in particular: **γάρ does not frame-forward; it strengthens-backward.**

| Marker | Discourse function | Direction |
|---|---|---|
| **ἰδού** | Deictic / mirative pointer | Forward — introduces a new on-stage element |
| **διό** | Inferential conclusion | Forward — draws an inference from prior |
| **οὖν** | Resumptive / consequential | Forward — advances closely from prior |
| **ἀλλά** | Adversative correction | Forward — corrects an expectation prior set |
| **γάρ** | Explanatory ground | **Backward** — strengthens prior assertion |
| **πλήν** | Restrictive adversative | Forward — narrows from prior |
| **τοιγαροῦν** | Strong inference | Forward — concludes strongly from prior |
| **νῦν δέ** (two-token) | Discourse pivot | Forward — pivots from hypothetical to actual |

## Source citations

- **Runge 2010 §2** — entire chapter "Connecting Propositions" surveys the NT Greek connectives matrix on three feature axes: ±continuity, ±development, ±support. Runge's §2.7 places γάρ at *+continuity, −development, +support* — the matrix slot for backward-strengthening rather than forward-development.

- **Levinsohn 2000 §5.4.2** — "Background Material with Γάρ." Verbatim: *"The presence of γάρ constrains the material that it introduces to be interpreted as strengthening some aspect of the previous assertion, rather than as distinctive information."* Levinsohn's finer empirical statistics corroborate Runge's matrix-slot assignment.

- **BDF §§438–470** — particles and conjunctions; primary scholarly source for the framing-device class as a coherent category. Provides the philological-historical grounding the discourse grammars build on.

## Why the colometric rule is uniform despite functional divergence

The colometric placement constraint (lead-content, never line-final) is a Layer-1-adjacent break-legality fact about Koine syntax: these particles attach proclitically or as clause-second elements to their host clause, and trailing them at line-end orphans the marker from its host. The discourse-direction divergence (forward-development vs. backward-strengthening) is an **information-structure** property orthogonal to the line-break legality property. Both layers must be respected; canon §3.3's R8 codifies only the colometric layer. The functional table above informs human-readable downstream interpretation but does not alter the rule.
