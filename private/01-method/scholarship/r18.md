# R18 — Vocative Scholarship

Substrate for canon §3.9 (R18 Vocative Rule, refined three-way treatment). The operational rule lives in the canon entry; this document carries the grammatical-grounding analysis and canonical cases the rule-template forbids inline.

## Why the apposition exception requires two justifications, not one

The default vocative-own-line rule treats a vocative as a discourse-turn-opening address act — the speaker turns toward an addressee, producing a new ATU. The apposition exception fires only when the vocative is grammatically appositive to an already-established second-person element in the same clause-span. Two distinct grammatical patterns satisfy this:

### Subject-appositive (vocative = implicit subject of 2p finite verb)

When the vocative names the implicit subject of a 2nd-person finite verb in the same clause, the verb and vocative form *one atomic predication*. The verb's morphology already encodes the second-person reference; the vocative is naming the addressee that the verb morphology presupposes.

Canonical cases:
- `Ὕπαγε, Σατανᾶ` (Matt 4:10) — 2p imperative + appositive vocative.
- `Ἀγνοεῖτε, ἀδελφοί` — 2p indicative + appositive vocative.
- `Θάρσει, τέκνον` (Mark 2:5) — 2p imperative + appositive vocative.
- `Οὐαὶ ὑμῖν, γραμματεῖς καὶ Φαρισαῖοι ὑποκριταί` — "woe to you" + appositive vocative restatement.

### Object-appositive (vocative restates explicit 2p pronoun)

When the vocative restates an explicit 2p pronoun already in the clause, the address is already established; the vocative is affective restatement.

Canonical cases:
- `Παρακαλῶ ὑμᾶς, ἀδελφοί` — explicit ὑμᾶς + appositive vocative.
- `Γνωρίζω ὑμῖν, ἀδελφοί` — explicit ὑμῖν + appositive vocative.

## Why verb person is irrelevant

The triggering condition for the apposition exception is the **2p pronoun**, not the person of the main verb.

- Luke 9:61 `Ἀκολουθήσω σοι, κύριε·` — `κύριε` merges because it is object-appositive to `σοι`, regardless of `ἀκολουθήσω` being 1st-person.
- Contrast Acts 9:10 `Ἰδοὺ ἐγώ, / κύριε.` — correctly split: `ἐγώ` is 1p, `κύριε` is not appositive to any 2p element in the clause-span.

This is why the rule names "2p pronoun" rather than "2p verb" as the structural signature.

## Boundary cases that stay on their own line

The default vocative-own-line rule re-asserts itself in these cases (no apposition trigger fires):

- **Turn-initial vocative** — opening a new discourse turn (paragraph-opening, post-speech-intro position). No preceding grammar to lean on within the turn.
- **Tail vocative** — clause-final address as a distinct rhetorical act.
- **Vocative after speech-intro punctuation (`·` / `:`)** — the prior 2p markers belong to the outer layer (the speech-intro frame), not the inner address. The speech-boundary punctuation severs the apposition relation.
- **Stacked parallel vocatives** — `πατέρες / νεανίσκοι / παιδία` (1 John 2:12–14) treated as a parallel address structure; J1 governs.

## Discourse-frame + vocative cluster

When a sentence-initial discourse marker (`Λοιπόν`, `Τὸ λοιπόν`, `Λοιπὸν οὖν`, `Ταῦτα δέ`, `Κἀγώ`, etc.) co-occurs with a vocative, *both* are extra-clausal elements and cluster on one line; the proposition follows on the next line.

Canonical: `Λοιπόν, ἀδελφοί,` / `στήκετε ἐν κυρίῳ.` (Phil 4:1 pattern). The vocative still earns its line — it earns *that* line together with the frame particle.

## Repeated vocatives stay together

`Κύριε κύριε` (Matt 7:21–22, Luke 6:46) is a single speech act, treated as one rhetorical unit.

## Burton + Wallace + BDF grounding

- **Burton §82–84** (vocative case) — vocative as direct-address syntactic category.
- **Wallace pp.65–71** (vocative case syntax) — modern complement to Burton; treats apposition cases and direct-address discipline.
- **BDF §145–149** (vocative) — philological grounding for the apposition relation between vocative and 2p pronoun / verb in NT Greek.
