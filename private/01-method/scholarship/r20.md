# R20 — Participial Phrase Scholarship

Substrate for canon §3.10b (R20 Participial Phrase, refined). The operational rule lives in the canon entry; this document carries the grammatical-grounding citations, the supplementary-vs-circumstantial-vs-attributive participle taxonomy, and the canonical examples that the rule-template forbids inline.

## Burton + Wallace + BDF + Smyth grounding

- **Burton §434–450** — adverbial-participle classes that R20/R22 discipline navigates: time, condition, concession, cause, purpose, means, manner, attendant circumstance. Burton's "attendant circumstance" maps closely to the supplementary-predication R20 split case.
- **Burton §445–446** — supplementary participles completing specific verbs (τυγχάνω, λανθάνω, φθάνω, παύω). The technical "supplementary" category in classical grammar.
- **Wallace §645–647** — modern complement to Burton on supplementary participles; same technical category framing.
- **Smyth §2094–96** — supplementary participles with verbs of beginning, ceasing, persisting, etc. The Greek-grammar-tradition canonical statement.
- **BDF §415–425** — participle syntax (attributive / circumstantial / supplementary distinction); philological foundation.

## Why our R20 usage is broader than "supplementary"

The grammarians' "supplementary participle" is a narrow technical category (completing specific verbs like τυγχάνω/λανθάνω/φθάνω/παύω). R20's *split-case* — the second-predication / ellipsis-of-main-verb test — is broader: ANY circumstantial participle whose semantic weight equals an independent predication via ellipsis of the main verb qualifies for split treatment. This includes:

- **Attendant-circumstance participles** (Burton's category) where the participle adds an additional action coordinate with the main verb.
- **Periphrastic-ellipsis cases** where an auxiliary `εἰμί` is implicit from a prior periphrastic explicit construction (see Acts 8:28 showcase below).
- **Postpositive predicate participles** that name a state coordinate with the main verb (e.g., `hestos` after `ophthe`).

The narrow "supplementary" category is a strict subset of what R20 splits.

## The second-predication test (worked examples)

**Question:** Can you reconstruct the participle as an independent predication by supplying the main verb? If yes → split. If the participle requires the main verb to become a thought at all → merge.

### Split examples (supplementary predication)

- **Luke 1:11** — `ὤφθη δὲ αὐτῷ ἄγγελος κυρίου / ἑστὼς ἐκ δεξιῶν τοῦ θυσιαστηρίου` — "an angel appeared" / "[appeared] standing at the right of the altar." The implicit "[appeared]" or "[was]" gives the participle a standalone predication. **Split.**
- **Mark 4:38** — `καὶ αὐτὸς ἦν ἐν τῇ πρύμνῃ / ἐπὶ τὸ προσκεφάλαιον καθεύδων` — "he was in the stern" / "[was] sleeping on the cushion." Periphrastic ellipsis; the second line shares `ἦν` with the first. **Split.**

### Merge examples (dependent frame)

- **Matt 2:10** — `ἰδόντες δὲ τὸν ἀστέρα ἐχάρησαν χαρὰν μεγάλην σφόδρα` — "having seen the star" is causal framing for "they rejoiced." The participle has no independent predication; supplying any auxiliary doesn't produce a standalone thought. **Merge.**
- **Luke 1:19** — `Ἐγώ εἰμι Γαβριὴλ ὁ παρεστηκὼς ἐνώπιον τοῦ θεοῦ` — attributive participle as title. Functions like a relative clause modifying `Γαβριήλ`. **Merge.**

## Periphrastic-ellipsis showcase (Acts 8:28)

When a periphrastic chain extends across multiple imperfective actions sharing one auxiliary, the auxiliary can be elided for subsequent elements. Each "ellipsized periphrastic" is a complete predication via ellipsis and stacks as its own line:

```
ἦν τε ὑποστρέφων                       ← explicit periphrastic ("he was returning")
καὶ καθήμενος ἐπὶ τοῦ ἅρματος αὐτοῦ    ← elliptical periphrastic ([ἦν] sitting)
καὶ ἀνεγίνωσκεν τὸν προφήτην Ἠσαΐαν.   ← imperfect equivalent, stacks as parallel
```

This is structurally a J1 (formally-marked parallel series) interaction: each line carries a complete predication via the shared / elided auxiliary.

## Boundary with R19 (genitive absolute) and R22 (orphaned-adverbial completion)

- **R19 (gen abs):** always own-line; structurally independent (has its own subject in genitive and its own predicate). R19 fires first.
- **R20 (circumstantial participle):** default merge with main verb; second-predication exception splits.
- **R22 (orphaned-adverbial completion):** specific override for short ὁπού/ὅτε/ὅς clauses that COMPLETE the predicate. Applies only to subordinate clauses, not participial constructions.

R20's split-default-merge / exception-split mirror is structurally the inverse of R22's split-default / completion-merge — both are calibrations of "does the dependent element constitute a separate ATU?"
