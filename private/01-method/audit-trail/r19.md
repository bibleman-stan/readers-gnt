# R19 — Audit Trail

Sweep history for canon §3.10a (R19 Genitive Absolute Always Own Line). The operational rule lives in the canon entry; this document carries sweep records and audit-precedent narratives.

## Sweep status

R19 has been one of the longest-stable rules in the canon — the "gen abs always own line" rule predates most of the other mechanical detectors. The applier `scripts/sweep_r19_genabs.py` has been run iteratively as new violations surfaced during cascade edits.

- **Acts 1:9 showcase analysis** — the canonical demonstration case for R19 + framework §1.1 bidirectional ATU test. The gen abs `βλεπόντων αὐτῶν` between the speech-act frame and the ascension main clause is the cleanest single-verse showcase of why R19's own-line rule is needed.
- **Anaphoric gen abs cluster (e.g., `Ταῦτα δὲ αὐτοῦ εἰπόντος,`)** — these were originally flagged by the standalone-ATU test as M3 fragments (failing forward-looking standalone test), but the bidirectional test codified at framework §1.1 (2026-05-13) resolved them as legitimate ATUs satisfying the anaphoric direction. R19 own-line still applies; the bidirectional test handles the standalone-ATU question separately.

## Codification provenance

The anaphoric-direction handling was codified after the 2026-05-13 framework §1.1 update on bidirectional ATU test. Prior to that, anaphoric gen abs (`Ταῦτα δὲ αὐτοῦ εἰπόντος`, etc.) were occasionally flagged as M3 fragments — the standalone-ATU test in only-cataphoric mode produced false-failures. The bidirectional test resolved this; canon §3.10a now treats both directions identically at the R19 line-placement layer.
